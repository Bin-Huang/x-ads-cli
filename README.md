# x-ads-cli

X Ads CLI designed for AI agents. Wraps the official X Ads API with simple, agent-friendly commands.

**Works with:** OpenClaw, Claude Code, Cursor, Codex, and any agent that can run shell commands.

> **Status: Under Development** — This project is not yet functional. Star/watch for updates.

## API Reference

- Official docs: https://developer.x.com/en/docs/x-ads-api
